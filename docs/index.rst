pygwtf: Gravitational waves in the time-frequency domain
========================================================

``pygwtf`` provides tools for constructing and analysing gravitational wave signals in the time-frequency domain.

If you find this package useful, please consider citing both the `software <https://zenodo.org/records/20531364>`_ and the `associated paper <https://arxiv.org/abs/2510.19047>`_.
